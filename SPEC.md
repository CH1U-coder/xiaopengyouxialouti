# 小朋友下樓梯 (NS-Shaft Clone) 遊戲規格書 (Game Spec)

這是一份關於垂直捲軸遊戲「小朋友下樓梯」同款遊戲的技術規格與實作計畫。

## 1. 遊戲核心機制 (Core Gameplay)
- **垂直捲軸**：畫面會持續向上滾動，玩家必須不斷向下跳到新的地板上。
- **生命值 (HP)**：
    - 玩家起始 HP 為 100。
    - 觸碰天花板的釘子或掉出畫面底部會扣血。
    - 踩到普通地板可緩慢回復 HP（或依經典規則不回復，僅避開傷害）。
- **得分**：根據下降的層數計分。

## 2. 玩家控制 (Player Control)
- **移動**：使用鍵盤左右方向鍵 (Left/Right Arrows)。
- **物理**：
    - 具備重力加速度。
    - 水平移動具有慣性與摩擦力（這會讓操作更有手感）。
    - 落在地板上會停止下墜。

## 3. 地板類型 (Platform Types)
- **普通地板 (Normal)**：最基本的平台，供玩家停留與下降。
- **釘子地板 (Spike)**：表面帶有尖刺，踩到的話玩家會受傷扣血。
- **彈簧地板 (Spring)**：踩到後會讓玩家高高彈起。
- **輸送帶 (Conveyor)**：分為向左與向右，玩家站在上面時會被強行推動。
- **易碎地板 (Fragile)**：玩家踩上去後約 0.5 秒會消失，無法長留。

## 4. 遊戲介面 (UI/UX)
- **主畫面**：顯示得分 (Floors) 與剩餘 HP 長條圖。
- **結束畫面**：顯示最終得分，並提供「重新開始」按鈕。
- **視覺風格**：雖然初期以色塊代替圖片，但整體配色會採用現代化的深色模式 (Dark Mode)，呈現科技感。

## 5. 技術架構 (Technical Stack)
- **HTML5 Canvas**：用於渲染遊戲畫面，確保在瀏覽器中流暢執行。
- **Vanilla JavaScript**：不使用重型引擎，直接用原生 JS 寫邏輯，輕巧且快速。
- **CSS3**：用於 UI 排版與轉場特效。
